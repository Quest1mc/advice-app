# Simple React app  that Gives random pieces of advice
## to show usage of state management and simple network call using axios

![alt text](https://github.com/Quest1mc/advice-app/blob/master/advice-app.png "Screenshot of Advice App")


The search box will allow you to type in a word and search for any quotes in the database that have said word.
the random advice button will give you a single pice of advice random context 
the lots of advice will look for 10 pieces of advice random context.
